# 🎭 Генератор промптов: Драматические истории

Веб-генератор промптов для создания драматических историй о женских судьбах, предательстве и торжестве справедливости. Подходит для ведения каналов на Дзене, Telegram, других платформах для женской аудитории 35-65 лет.

## 🌐 Демо

[Открыть генератор](https://your-username.github.io/drama-prompt-generator/)

## ✨ Возможности

- **7 параметров** для тонкой настройки истории:
  - Тип героини (15 вариантов)
  - Тип антагониста (15 вариантов)
  - Вид предательства (12 вариантов)
  - Способ открытия правды (10 вариантов)
  - Что героиня отдала/потеряла (10 вариантов)
  - Финал истории (8 вариантов)
  - Длина рассказа (3 варианта)

- **Случайный выбор** для быстрой генерации
- **Копирование в буфер обмена** одним кликом
- **Открытие в Gemini** для немедленной отправки промпта
- **Адаптивный дизайн** для всех устройств

## 📋 Как использовать

1. Откройте генератор в браузере
2. Выберите параметры истории из выпадающих списков
3. Нажмите **«Сгенерировать промпт»**
4. Скопируйте готовый промпт кнопкой **«Скопировать в буфер»**
5. Вставьте промпт в Gemini, ChatGPT, Claude или другую LLM
6. Получите готовый рассказ для публикации

## 🚀 Быстрый запуск через GitHub Pages

1. Форкните этот репозиторий
2. Зайдите в **Settings** → **Pages**
3. В разделе **Source** выберите ветку `main` и папку `/ (root)`
4. Сохраните. Через 1-2 минуты ваш генератор будет доступен по адресу:
   `https://ваш-логин.github.io/drama-prompt-generator/`

## 🎯 Целевая аудитория контента

- Женщины 35-65 лет
- Платформы: Дзен, Telegram, ВКонтакте, личные блоги
- Тематика: семейные драмы, предательство, справедливость, женская сила

## 📝 Структура генерируемых историй

Каждый промпт включает обязательную структуру:

1. **Цепляющий заголовок** от первого лица
2. **Завязка** — описание обычной жизни героини
3. **Появление проблемы** — антагонист с маской добродетели
4. **Эскалация** — нарастание конфликта
5. **Момент истины** — случайное открытие правды
6. **Подготовка** — холодный сбор доказательств
7. **Кульминация** — драматическое разоблачение
8. **Развязка** — торжество справедливости
9. **Эпилог** — новая жизнь героини спустя время

## 🎨 Стилистические особенности промптов

- Живые диалоги с яркими репликами
- Детальный быт (бренды, цены, запахи, звуки)
- Контраст эмоциональных состояний "до" и "после"
- Реалистичные детали (суды, банки, полиция)
- Эмоциональная дуга героини: терпение → шок → сила
